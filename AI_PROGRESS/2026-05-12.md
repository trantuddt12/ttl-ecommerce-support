### 2026-05-12 22:40 - Backend - Fix order detail multiple bag fetch

- Scope: backend
- Changed:
  - `ecommerce-shop/basecommerce/src/main/java/com/ttl/base/repositories/OrderRepository.java`
- Summary:
  - Fixed `MultipleBagFetchException` on `GET /orders/me/{id}` by removing `histories` from `OrderRepository` entity graphs that already fetch `items`. `Order.items` and `Order.histories` are both `List` bags, so Hibernate cannot join-fetch both in the same query.
